第1章 PowerQuery入门

课时1 为什么学习PowerQuery

PowerQuery是什么？

  • 优质的 ETL工具（Extract-Transform-Load）

  • 数据源 -> 抽取 -> 转换 -> 加载 -> 目的端（Excel / PowerBI）-> 数据分析 -> 数据可视化

PowerQuery 优势

  • 可连接 上百种数据源，解决了数据源分散的问题
  • 无需编程，也可对数据快速清洗
  • 支持 大量数据 的快速处理
  • 记录所有操作步骤 -> 数据源 数据变化 -> 只需 刷新 即可（无需重复操作）

PowerQuery在哪里？

  • PowerQuery，直接植入到 Excel / PowerBI，非常易于获取和使用

  • 直接打开方式
  • 新建查询打开
  • 已有查询打开

Excel 不同版本 差异？

  • 2016
  • 2010 / 2013 -> 需安装PowerQuery插件（下载链接）

课时3 下载和安装PowerBI

什么是 PowerBI？

  • 微软提供的一套 免费 的 商业分析工具
  • PowerQuery（数据清洗）、Power Pivot（数据分析）、Power View（数据可视化）

下载 & 安装 PowerBI？

  • 微软 应用商店
  • 网页 下载安装（下载地址）
  • 及时更新PowerBI 新功能
